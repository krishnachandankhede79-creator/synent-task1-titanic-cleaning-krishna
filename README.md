# 🚢 Titanic Data Cleaning Project

## 📌 Problem Statement
The goal of this project is to clean and preprocess the Titanic dataset.

## 📊 Dataset
Titanic dataset in CSV format.

## ⚙️ Steps Performed
- Handled missing values (Age filled with median, Embarked with mode)
- Removed duplicate rows
- Dropped Cabin column due to many missing values
- Converted data types (Survived to integer)
- Renamed columns (Pclass → Passenger_Class, Sex → Gender)

## ✅ Result
Clean dataset ready for analysis.
